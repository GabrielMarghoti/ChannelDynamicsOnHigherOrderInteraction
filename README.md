# Impact of Channel Dynamics on Higher-order Interactions of Oscillators

## Overview

This project establishes a theoretical/computational framework for evaluating the use of phenomenological higher-order interactions (HOIs) in oscillator networks coupled by channels. We demonstrate analytically that standard 3-body Kuramoto couplings are mathematically equivalent to underlying pairwise connections modulated by latent transmission variables in the adiabatic limit.

## Scientific Rationale

The prevalent reductionist approach maps complex synchronization transitions to static hyperedges. However, relying exclusively on static structural hypergraphs obscures fundamental physical asymmetries and non-equilibrium transmission inertia. Instead of treating interactions as instantaneous static weights, this model utilizes a dynamical state variable, $u_{ij}(t)$, to represent the actual propagated signal (or effective interaction current) subject to a finite relaxation timescale $\tau$.

## Key Findings

* **The Adiabatic Limit:** Structural multi-body interactions naturally emerge as an artifact when transmission dynamics are assumed to be infinitely fast ($\tau \ll 1$) compared to macroscopic phase evolution.


* **Symmetry Constraints:** The standard trigonometric $(1,1,-2)$ 3-body phase coupling strictly requires the local mean-field interference to be perfectly isotropic ($B_{ijk} = B_{ikj}$). Real-world spatial embedding and directed pathways naturally break this reciprocity.


* **Dynamical Emergence:** Relaxing the adiabatic constraint reveals that finite transmission inertia drives distinct macroscopic states. Transmission delays induce bistability for symmetric interaction tensors and anti-phase cluster synchronization for antisymmetric ones.


* **Finite-Size Scaling:** The cluster synchronization observed in globally coupled antisymmetric topologies is fundamentally a finite-size effect; mean-field dominance washes out local delayed feedback in dense limits ($N \gtrsim 100$).
